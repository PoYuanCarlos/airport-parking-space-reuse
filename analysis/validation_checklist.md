# Validation Checklist

- [ ] Confirm public/synthetic data boundary.
- [ ] Check row counts before and after cleaning.
- [ ] Check duplicate keys.
- [ ] Check unmatched records after joins.
- [ ] Check whether utilization ratios are between 0 and 1 unless intentionally indexed.
- [ ] Check whether scenario assumptions are clearly labeled.
- [ ] Check that dashboard conclusions do not overstate prediction accuracy.
