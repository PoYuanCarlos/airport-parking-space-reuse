# Risk Register

| Risk | Impact | Control Action |
|---|---|---|
| Public data unavailable | Medium | Use clearly labeled synthetic data for first release. |
| Scenario assumptions overinterpreted | High | Label assumptions and avoid prediction language. |
| Dashboard too complex | Medium | Use a decision-support layout with limited views. |
| Privacy/copyright exposure | High | Review all files, images, and data before upload. |
| Technical overclaiming | High | Describe the project as prototype, concept, or portfolio demonstration. |
