# Scope Statement

Analyze scenario-based airport parking demand reallocation under shared autonomous mobility and translate results into a public-facing decision-support dashboard and portfolio case-study page.

## In Scope

- Public/synthetic data design
- Scenario assumptions
- Parking utilization indicators
- Curbside/staging pressure proxy
- Reuse-option scorecard
- Tableau dashboard prototype
- GitHub Pages case-study page

## Out of Scope

- Production data pipeline
- Real-time airport operations integration
- Confidential airport records
- Exact AV market forecast
