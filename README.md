# Airport Parking & Space Reuse Strategy Dashboard

Scenario-based transportation analytics portfolio project evaluating airport parking demand reallocation and space reuse under shared autonomous mobility.

## Project Summary

This project evaluates how shared autonomous mobility may reduce long-term parking demand, increase curbside and staging pressure, and create opportunities for adaptive reuse of airport parking space.

The project is designed as a decision-support portfolio case study, not only a dashboard exercise.

## Core Question

How can airports evaluate the operational, financial, spatial, and stakeholder impacts of shared autonomous mobility on parking demand, curbside pressure, and future parking-space reuse under uncertain adoption scenarios?

## Method: P.A.C.E.

- **P — Problem Framing:** airport access problem, stakeholders, scope, assumptions, and KPIs.
- **A — AI-Assisted Data Processing:** BigQuery SQL workflow for importing, cleaning, joining, and validating data.
- **C — Computational Analysis + EDA:** scenario modeling, exploratory analysis, sensitivity checks, and dashboard preparation.
- **E — Evidence-Based Decision Support:** translate analysis into recommendations, limitations, and next actions.

## Tool Stack

- GitHub Pages for the public project website
- BigQuery SQL for data preparation and validation
- Tableau for dashboard visualization
- Excel / Python as needed for support analysis
- AI-assisted workflow review with human validation
